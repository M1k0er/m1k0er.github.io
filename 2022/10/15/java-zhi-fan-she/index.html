java之反射

编程 java
编程
发布日期:   2022-10-15

0x00 前言

旧梦很美，虽未能实现，但我很欣慰它们曾萦绕心田。

0x01 什么是反射

反射就是Reflection，Java的反射是指程序在运行期可以拿到一个对象的所有信息。有点java基础的人都知道如果我们要调用一个对象的方法或者属性，通过会传入一个实例，比如以下代码块：

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        User u = new User();
        u.setUsername("张三");
        System.out.println(u.getUsername());
    }
}

以上代码基于你能获取User类的情况下，那如果没法获取User类能不能实现以上代码的功能呢，答案是可以的，这里我们讲的反射就是为了解决在运行期，对某个实例一无所知的情况下，如何调用其方法和属性。

0x02 Class类

在java中，除了int基本类型外，其他类型全部是class(包括interface)，而class是由JVM在执行过程中动态加载的，每加载一种class，JVM就为其创建一个Class（即一个class在内存中只有一个Class对象），这里的Class类型是一个名叫Class的class：

public final class Class {
    private Class() {}
}

这个Class实例是由JVM创建的，我们自己编写的java程序是无法创建Class实例的。

由上所述，我们很容易知道一个Class实例包含了该class的所有完整信息，包括类名、包名、父类、实现的接口、类的属性、方法等，这种通过Class实例获取class信息的方法称为反射。

0x03 使用方法

1、获取class的Class实例

1）、通过静态变量class获取

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        Class userClass = User.class;
        System.out.println(userClass);
    }
}

2）、通过getClass()方法获取

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        User user = new User();
        Class aClass = user.getClass();
        System.out.println(aClass);
    }
}

3)、通过静态方法Class.forName()获取(前提需要知道完整的包名)

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        Class aClass = Class.forName("org.root.pojo.User");
        System.out.println(aClass);
    }
}

2、类名的获取

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        Class aClass = Class.forName("org.root.pojo.User");//获取对象
        String name = aClass.getName();
        String simpleName = aClass.getSimpleName();
        System.out.println("通过getName()获取:"+name);
        System.out.println("通过getSimpleName()获取:"+simpleName);
    }
}

3、类属性的获取

Class提供了以下几个方法来获取属性

  • Field getField(name)：根据字段名获取某个public的属性
  • Field getDeclaredField(name)：根据字段名获取当前类的某个field（不包括父类）
  • Field[] getFields()：获取所有public的属性（包括父类）
  • Field[] getDeclaredFields()：获取当前类的所有field（不包括父类）

示例代码：

package org.root;
import org.root.pojo.User;
import java.lang.reflect.Field;

public class Main {
    public static void main(String[] args) throws ClassNotFoundException, NoSuchFieldException {

        Class aClass = Class.forName("org.root.pojo.User");//获取对象
        Field user = aClass.getField("user");
        System.out.println("getField获取的信息:"+user);
        Field[] fields = aClass.getFields();
        for (Field field : fields) {
            System.out.println("getFields()获取的信息:"+field);
        }
        Field username = aClass.getDeclaredField("username");
        System.out.println("getDeclaredField()获取的信息:"+username);
        Field[] declaredFields = aClass.getDeclaredFields();
        for (Field declaredField : declaredFields) {
            System.out.println("getDeclaredFields()获取的信息:"+declaredField);
        }
    }
}

当我们获取了Field对象后，我们就能通过这个Field对象获取字段的所有信息：

  • getName()：返回字段名称，例如，"name"；
  • getType()：返回字段类型，也是一个Class实例，例如，String.class；
  • getModifiers()：返回字段的修饰符，它是一个int，不同的bit表示不同的含义。

4、获取class的方法

Class类提供了以下几个方法来获取class的方法

  • Method getMethod(name, Class...)：只能获取public的指定Method
  • Method getDeclaredMethod(name, Class...)：获取当前类的指定Method（包括public和private）
  • Method[] getMethods()：获取所有public属性的Method
  • Method[] getDeclaredMethods()：获取当前类的所有Method（包括public和private）

示例代码：

package org.root;
import org.root.pojo.User;

import java.lang.reflect.Field;
import java.lang.reflect.Method;

public class Main {
    public static void main(String[] args) throws ClassNotFoundException, NoSuchFieldException, InstantiationException, IllegalAccessException, NoSuchMethodException {

        Class aClass = User.class;
        Method method = aClass.getMethod("getUsername");
        System.out.println("getMethod方法："+method);
        Method[] methods = aClass.getMethods();
        for (Method method1 : methods) {
            System.out.println("getMethods方法："+method1);
        }

        Method toString = aClass.getDeclaredMethod("toString");
        System.out.println("getDeclaredMethod方法"+toString);

        Method[] declaredMethods = aClass.getDeclaredMethods();
        for (Method declaredMethod : declaredMethods) {
            System.out.println("getDeclaredMethods方法"+declaredMethod);
        }
    }
}

5、调用构造器

通常来说，我们创建一个对象可以用以下代码来实现

User user = new User();

通过反射创建实例，我们可以调用newInstance()方法：

Class aClass = User.class;
User o = (User) aClass.newInstance();

调用newInstance()方法也是有限制的，它需要满足两个条件

  • 类必须有一个无参的构造器
  • 类的构造器访问权限需要足够

那么是不是没有无参构造器就不能创建对象了呢，答案是否定的，只要操作的时候明确调用类中的构造器，并将参数传递进去之后，才可以实例化操作。我们可以通过以下代码来看看具体的实现。

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        Class aClass = User.class;
        User o = (User) aClass.newInstance();//创建对象
        //通过构造器创建对象
        Constructor constructor = aClass.getDeclaredConstructor(int.class, String.class, String.class);
        User o1 = (User) constructor.newInstance(1, "李强", "1234565");
        System.out.println(o1);
    }
}

简单说下上面代码的实现逻辑，首先通过Class的getDeclaredConstructor方法取得本类的指定形参类型的构造器，然后向构造器的形参中传递一个对象数组进去，里面包含了构造器中所需的各个参数，最后通过Constructor实例化对象。

6、调用方法

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        Class aClass = User.class;
        User o2 = (User) aClass.newInstance();//创建对象
        Method setUsername = aClass.getDeclaredMethod("setUsername", String.class); //获取方法
        setUsername.invoke(o2,"李强");//通过invoke(对象，参数)    调用方法
        System.out.println("通过反射调用方法:"+o2.getUsername());
    }
}

7、调用属性

package org.root;
import org.root.pojo.User;

public class Main {
    public static void main(String[] args){
        Class aClass = User.class;
        User o3 = (User) aClass.newInstance();//创建对象
        Field username = aClass.getDeclaredField("username");
        username.setAccessible(true);
        username.set(o3,"李强");
        System.out.println("通过反射操作属性:"+o3.getUsername());
    }
}

这里重点讲下setAccessible()这个方法，因为这里的username是私有的，我们没办法直接调用它，因此需要通过setAccessible()来取消安全检测，允许访问私有属性，默认是false，私有方法也是一样：方法..setAccessible(true)
